---

## 四份檔案各是什麼

| 檔名 | 形式 | 給誰看 | 怎麼看 |
|---|---|---|---|
| `三小時流程教案.md` | 教案總覽 | 學員 & 講師 | 想知道**今天會做什麼**、時間怎麼分、哪段教什麼 → 看這份 |
| `完整實況版教學手冊.md` | 課堂實況版 | 學員 | 想看**江江當天現場說了什麼**（金句、學員提問、現場示範）→ 看這份。等於坐在現場聽過一遍 |
| `三小時投影簡報.html` | 課堂簡報 | 學員 | 用瀏覽器打開（雙擊或拖進 Chrome），就是課堂上播的那份簡報。**叫 AI 照這份做也可以** |
| `總結心智圖.html` | 一頁總覽 | 學員 | 用瀏覽器打開，整堂課的視覺化心智圖。複習用 |

---

## 怎麼搭配 AI 用

對 Codex / Claude Code 說：

- 「**幫我照著 docs/三小時投影簡報.html 跟我講一遍**」 → AI 會逐頁帶你
- 「**docs 裡這份教學手冊有提到 XXX 嗎？**」 → AI 直接 grep + 回答
- 「**我看完了，我想做 XXX，幫我從哪裡開始？**」 → AI 對照教學手冊找對應段落

---

## 跟根目錄的關係

- 根目錄 `README.md` = 給「**還沒下載**」的人看（一分鐘啟動指引）
- 本資料夾 `docs/` = 給「**已經下載完**」的學員看（教材本體）
