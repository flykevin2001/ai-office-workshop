# 工作助理

這個資料夾是個人工作助理系統。日常目標很簡單：每天快速記錄、專案分開管理、早上能看到今日安排、週末能依樣板產出報告。

## 每天怎麼用

1. 進入本資料夾後輸入 `今日工作`，小K會讀取 `01_專案/` 內所有專案，依待辦壓力排序。
2. 輸入 `記錄進度：...`，小K會把內容整理到 `02_每日紀錄/YYYY-MM-DD.md`，並同步更新相關專案檔案。
3. 有新專案時輸入 `新增專案：專案名稱`，小K會依 `04_資源/專案資料夾模板.md` 建立資料夾。
4. 每週輸入 `產出本週週報`，小K會依 `03_週報/templates/通用週報樣板.md` 匯整本週內容。

## 核心資料夾

- `00_今日工作/`：每日入口、任務總表、本週摘要。
- `01_專案/`：每個專案一個資料夾，資料、輸出、狀態分開放。
- `02_每日紀錄/`：每日輸入與AI整理後的工作紀錄。
- `03_週報/`：週報樣板與每週輸出。
- `04_資源/`：模板、規則、常用參考。
- `09_封存/`：結案或半年以上不常用的資料。
- `outputs/`：跨專案輸出檔。

## 可執行工具

更新今日工作與任務總表：

```powershell
.\tools\update-dashboard.ps1
```

產出本週 Markdown 週報，若本機有 `pandoc` 會同時輸出 DOCX/PDF：

```powershell
.\tools\export-weekly-report.ps1
```
