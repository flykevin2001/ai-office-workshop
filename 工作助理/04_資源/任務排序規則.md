# 任務排序規則

`今日工作` 的排序依下列欄位由高到低排序。

| 順序 | 欄位 | 說明 |
|---|---|---|
| 1 | 逾期待辦數量 | `due` 早於今天且未完成 |
| 2 | 今日到期待辦數量 | `due` 等於今天且未完成 |
| 3 | 專案優先級 | 高 > 中 > 低 |
| 4 | 未更新天數 | `最後更新日期` 距今天越久越前面 |
| 5 | 未完成待辦總數 | 未完成 checkbox 越多越前面 |

任務格式：

```markdown
- [ ] 任務內容 #專案ID priority:高 due:2026-06-24
- [x] 已完成任務 #專案ID done:2026-06-23
```
