# 專案資料夾模板

建立新專案時，複製以下結構。

```text
PROJECT-ID_專案名稱/
├── 專案狀態.md
├── 進度紀錄.md
├── 待辦事項.md
├── 重要事項.md
├── 資料/
└── outputs/
```

## 專案狀態.md

```markdown
# 專案狀態：專案名稱

- 專案ID：PROJECT-ID
- 狀態：進行中
- 優先級：中
- 負責人/窗口：
- 最後更新日期：YYYY-MM-DD

## 目前進度

- 

## 下一步

- 

## 風險與阻礙

- 

## 週報摘要

- 本週完成：
- 目前狀態：
- 下週計畫：
```

## 進度紀錄.md

```markdown
# 進度紀錄

## YYYY-MM-DD

- 做了什麼：
- 為什麼：
- 下一步：
```

## 待辦事項.md

```markdown
# 待辦事項

## 未完成

- [ ] 任務內容 #PROJECT-ID priority:中 due:YYYY-MM-DD

## 已完成

- [x] 已完成任務 #PROJECT-ID done:YYYY-MM-DD
```

## 重要事項.md

```markdown
# 重要事項

## 決策

- YYYY-MM-DD：

## 風險

- YYYY-MM-DD：

## 等待他人

- YYYY-MM-DD：
```
