# 週報欄位映射

| 樣板欄位 | 來源 |
|---|---|
| `{{week}}` | 系統日期計算的 ISO 週別 |
| `{{start_date}}` | 本週週一 |
| `{{end_date}}` | 本週週日 |
| `{{summary}}` | `02_每日紀錄/` 本週內容摘要 |
| `{{project_progress}}` | 各專案 `專案狀態.md` 的目前進度與週報摘要 |
| `{{completed_items}}` | 各專案 `待辦事項.md` 的本週完成項 |
| `{{risks}}` | 各專案 `重要事項.md` 的風險、阻礙 |
| `{{next_week_plan}}` | 各專案 `專案狀態.md` 的下一步 |
| `{{waiting_for}}` | 各專案 `重要事項.md` 的等待他人 |

如果 Kevin 提供正式週報樣板，先放入 `03_週報/templates/`，再更新這份映射表。
