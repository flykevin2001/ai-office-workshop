# 👋 從這裡開始

歡迎進入 **AI 辦公室**。
這是一份「給 AI 用的工作環境」——所有檔案都放在這個資料夾裡，AI 來「上工」就找這裡。

---

## ⚡ 一分鐘啟動

最簡單的版本——**跟 AI 說「幫我安裝」就好**：

1. 開個新資料夾（例如「**我的 AI 辦公室**」）
2. 開啟 Codex，cd 進去
3. 把這個 GitHub 連結貼給 AI，跟它說：

   > **「幫我安裝這個」**
   > `https://github.com/Jiang-Yude/ai-office-workshop`

4. AI 會自動把整套 AI 辦公室拷貝進來
5. 完成後跟 AI 說「**咪卡在嗎**」開始用

就這樣，跳過下載 ZIP 跟解壓縮。

---

### 不會用 GitHub 也沒關係

如果你完全不會用 GitHub，**雲端硬碟下載**也可以：

- 江江教練的學員：LINE 群裡有雲端硬碟連結，下載資料夾即可
- 一般人：到 GitHub 頁面點綠色按鈕「**Code → Download ZIP**」，解壓縮到電腦

## 第一句話跟 AI 說

拷貝下來後，跟 AI 說：

> **「咪卡在嗎」**

AI 會讀取 `agents.md`，認識自己的工作環境，然後出場。

如果 AI 沒按腳本念招呼語、或自編一個情境化開場——**沒關係**，你直接看下方「你可以選三條路」自己決定要從哪開始。

## 🐱 你可以選三條路

> 嗨，我是江江教練設計的 AI 助理。
> 歡迎進入 AI 辦公室。

跟 AI 對話時，三條路任選一條：

### 1. 🎯 現在就有想處理的文件或專案嗎？

→ 「**我有一份 [XXX]，幫我處理**」
→ AI 直接動手，一邊做一邊告訴你發生了什麼事。

### 2. 🗺️ 想先認識這裡有哪些助理可以幫你？

→ 「**這裡有哪些助理可以幫我？**」
→ AI 介紹一輪，讓你以後知道該叫誰。

### 3. 💭 想慢慢來也可以

→ 「**我想聊聊我手上的工作**」
→ AI 陪你聊，再設計適合你的方式。

## 📚 想升級到最新版？

說「**檢查更新**」或「**有沒有新版**」，AI 會自動檢查更新通道並引導你 OTA 更新。

---

## ⚠️ 注意事項

1. **這個資料夾本身就是你的 AI 工作環境**，不要刪、不要搬到 iCloud 外的奇怪地方
2. **macOS 放桌面或文件夾都可以**
3. **Windows 看到亂碼**請執行 `chcp 65001` 或改用 VS Code 開啟
4. **第一次跑可能會慢**——AI 在讀整個資料夾架構，正常

---

## 🆘 卡關了怎麼辦？

對 AI 說：

- 「**我卡住了**」→ AI 會問你卡在哪、幫你想辦法
- 「**幫我看看這個錯誤**」+ 貼出錯誤訊息 → AI 會幫你 debug
- 「**重新開始**」→ AI 會回到初始狀態

或者直接退出（`Ctrl+D` 或 `/quit`），重新進來再試。
