# 教學釋放檔案

這裡的東西是給你**第一次玩咪卡（Mika）**用的範例素材。

---

## 裡面有什麼

- **How-People-Use-Claude-Appendix.pdf**（24 頁）— Anthropic 官方研究：人們怎麼把 Claude 用在情感支持、建議、陪伴
- **Teams-Use-Claude-Code.pdf**（23 頁）— Anthropic 官方文件：團隊如何在實務中使用 Claude Code
- **截圖示範/** — 練習圖片轉檔、截圖改名

兩份 PDF 都是 **Anthropic 官方公開文件**，輕量、適合練手感（過去用納瓦爾寶典 200+ 頁太大了）。

---

## 用不到？整個刪掉沒關係

這個資料夾**整個刪掉，咪卡也能正常運作**。

刪法：
- 跟咪卡說「把教學釋放檔案這個資料夾刪掉」（咪卡會搬到 `_trash/` 備份，不是真的刪）
- 或自己拖到垃圾桶

## 第一次想玩什麼？跟咪卡這樣說

### 推薦的入門三步

**步驟 1：先讓咪卡摘要 PDF 看內容**
> 「咪卡，幫我摘要 教學釋放檔案/How-People-Use-Claude-Appendix.pdf，3 段就好。」

**步驟 2：請咪卡翻譯成繁體中文**
> 「幫我把這份 PDF 翻譯成繁體中文。」
> （可選：「給我簡化版」或「1:1 完整翻譯」）

**步驟 3：轉成 Markdown 存下來**
> 「轉成 Markdown 存到我桌面。」

跑完這三步，你就用過了：**摘要 → 翻譯 → 轉檔** 三個最常用的工具。

### 其他練習任務

| 你想試 | 跟咪卡說 |
|---|---|
| 改截圖檔名 | 「幫我看 教學釋放檔案/截圖示範/ 裡有什麼截圖，用中文重新命名」 |
| PDF 翻譯 | 「把 教學釋放檔案/How-People-Use-Claude-Appendix.pdf 翻譯第一章」 |
| PDF 轉 Markdown | 「把 教學釋放檔案/Teams-Use-Claude-Code.pdf 轉成 Markdown，按章節拆分」 |
| 跟納瓦爾聊天 | 「我想找納瓦爾聊聊財富自由」（用 naval-ravikant 技能包，不需要 PDF） |
