# 教學示範檔案

> 課堂練習用的素材。這次線上課只用到「**截圖改名**」這一份，所以這個資料夾現在只留截圖示範。

---

## 裡面有什麼

- **截圖示範 原始版/** — 一堆檔名亂七八糟的截圖（`截圖 2026-03-14 下午4.54.24.png` 那種），給你練「**改成看得懂的中文檔名**」用

---

## 為什麼要保留亂檔名

因為這堂課的重點之一是：**江江教練會在課堂上手把手帶你做出一個叫做 `image-renamer` 的技能包**。

做出來之後，你就會用「你親手做的技能包」改這些截圖的名字。這個資料夾就是讓你**有東西可以練**。

---

## 怎麼跟 AI 練習

課堂做完 `image-renamer` 之後，跟咪卡這樣說：

> 「幫我看 `教學示範檔案/截圖示範 原始版/` 裡有什麼截圖，用中文重新命名。」

咪卡會走你剛才做出來的 `image-renamer` 流程，把這些截圖一張一張改成「看得懂的事件 + 日期」格式。

---

## 用不到？整個刪掉沒關係

這個資料夾**整個刪掉，AI 辦公室也能正常運作**。是給課堂練習用的，課程結束你可以自己決定要不要留。

刪法：跟咪卡說「把教學示範檔案這個資料夾刪掉」（咪卡會搬到 `_trash/` 備份，不是真的刪）。
