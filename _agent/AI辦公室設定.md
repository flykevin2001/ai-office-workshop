## 新手指南

歡迎來到你的 AI 辦公室。對外產品形象暫定為 **Mika 貓咪管家**：一隻會幫你整理資料、看住工作台、提醒下一步的小管家。這裡是你的功能儀表板，可以看到所有功能的狀態。

### 第一次使用？

跟 AI 說「助理在嗎」或「開始初始化」，AI 會引導你完成以下設定：

1. 建立 `00 工作台/` 資料夾和工作看板
2. 建立垃圾桶和標籤字典
3. 介紹每個功能

### 已經初始化過了？

你可以隨時回來這裡：
- 勾選或取消功能
- 跟 AI 說「設定」就會打開這一頁
- 改了設定後，AI 下次啟動會自動讀取

## 核心功能（預設啟用）

這些是 AI 辦公室的基礎功能，建議保持開啟。

- [x] **AI 助理** — 單一入口：教你、帶你、幫你執行（原「AI 助教」合併進來）
- [x] **專案助理** — 看板、進度、任務追蹤
- [x] **辦公室秘書** — AI 辦公室建置、檔案管理、記日誌、安全保護
- [x] **工作看板** — 任務追蹤中心，AI 每天第一個讀的地方
- [x] **安全機制** — 垃圾桶 + 操作前確認 + 禁止直接刪除
- [x] **檔案命名規則** — 新檔案自動加日期時間戳
- [x] **三基本功教學** — 3X4 資料管理法、檔名命名、標籤管理（環環相扣，從眼前資料一點一滴開始）
- [x] **任務結束主動提問** — AI 助理會主動問要不要寫工作日記、要不要做技能包

## 建議啟用

這些功能可以讓你的 AI 辦公室更好用，建議用了一週後開啟。

- [ ] **標籤系統** — 幫文件加標籤，用標籤字典統一用詞
- [ ] **AI 工作日誌** — AI 每次操作後自動記錄，方便你回顧
- [ ] **每日待辦掃描** — 說「今天做什麼」自動掃看板回報
- [ ] **3X4 資料管理法** — 四種時效的檔案放置規則

## 格式轉檔助理（小工具）

格式轉檔助理由三個獨立的小工具組成，有需要再啟用。

- [ ] **截圖改名** — 把截圖的亂碼檔名改成中文（image-renamer）
- [ ] **圖片格式轉換** — HEIC 轉 JPG、PNG 轉 WebP 等（image-converter）
- [ ] **PDF 翻譯卡片化** — 翻譯 PDF 並整理成重點卡片（pdf-translation-cardifier v2.0）

## 文件整理助理

- [ ] **文件整理助理** — 幫文件想標籤、上標籤、改詳細檔名、管字典檔（tag-manager v1.0）

## 規劃中的助理

這些功能對外已公告，技能包尚未完成：

- [ ] **順稿助理** — 尊重你的風格，優化你的文案
- [ ] **盲點助理** — 幫你檢查思考盲點、邏輯漏洞
- [ ] **排班助理** — 幫小團隊、小部門或朋友協作整理人員、假期偏好、限制條件，產出可討論的班表草稿
- [ ] **搜尋助理** — 針對一個問題派出三個角度搜尋：官方資料、正面案例、負面風險

## 進階功能

這些需要特定環境支援，或適合有經驗的使用者。

- [ ] **Hook 防護** — 自動攔截危險操作（僅 Claude Code / Cowork）
- [ ] **受控詞彙** — 進階標籤管理，統一同義詞，建立受控詞彙表
- [ ] **多看板模式** — 從一個看板擴展成多個（發文、邀約、課程）
- [ ] **工作日記引導** — AI 用 What → Why → How → So What 引導你寫工作日記
- [ ] **記憶持久化** — 在換視窗、任務結束或壓縮前，把重要狀態寫成可接續文件
- [ ] **策略性壓縮** — 在研究結束、執行前或階段完成後，主動整理上下文再進下一段
- [ ] **持續學習** — 從重複流程、錯誤修正、使用者偏好中萃取操作手冊，必要時幫你編寫、更新 Skills
