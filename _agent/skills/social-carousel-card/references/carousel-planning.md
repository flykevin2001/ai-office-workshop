# 輪播規劃規則

## 平台與版型

| 使用情境 | 建議比例 | 建議張數 |
|---|---:|---:|
| IG + Threads 共用 | 4:5 直式 | 5-10 |
| Threads 快速觀點 | 4:5 直式 | 3-5 |
| IG 教學型輪播 | 4:5 直式 | 7-10 |
| 單張社群圖 | 1:1 正方形 | 1 |
| 一頁總結 / 簡報封面 | 16:9 橫式 | 1 |
| 高資訊密度受眾 | 4:5 或自訂 | 3-5 |
| 新手受眾 / 慢慢鋪陳 | 4:5 | 7-10 |

## 拆卡原則

- 一張卡只承擔一個任務：吸引、鋪陳、解釋、示範、總結、導流。
- 封面只講「為什麼要看」，不要塞完整論證。
- 內容卡要把抽象觀點變成例子、步驟、對照或清單。
- CTA 要符合使用者的引流方式：留言關鍵字、私訊、報名、下載、收藏、追蹤。
- 使用者若已定稿文案，拆卡只做結構安排，不擅自改語氣或結論。

## 生成前檢查

- 是否已確認比例。
- 是否已確認張數。
- 是否已確認文案可否微調。
- 是否有品牌語氣、受眾與 CTA。
- 是否要先出第 1 張試跑。
