# 逐張生成規則

## 硬規則

- 一次只生成一張卡。
- 不做拼圖、九宮格、多卡合併。
- 不在同一張圖裡塞 5 張卡。
- 每張都是完整獨立畫面。
- 圖上文字要可讀，不遮擋、不裁切。
- 若使用者指定品牌角色或素材，角色與素材不可搶主視覺。

## Prompt 結構

每張卡用這個格式：

```text
請生成一張社群輪播卡。
比例：[4:5 直式 / 1:1 正方形 / 16:9 橫式 / 自訂比例]
卡序：[第 N 張，共 M 張]
卡片目的：[封面 / 問題 / 核心觀念 / 步驟 / 案例 / CTA / 一頁總結]
主標：[文字]
內文：[文字]
版面：[單張主視覺 / 上下 2 區 / 左右 2 欄 / 流程 / 清單 / 對照表]
視覺元素：[角色、圖示、文件、便利貼、框線、圖表等]
品牌規則：[語氣、配色、Logo、角色、禁用詞]
輸出要求：只輸出這一張，不要把其他卡片合併進來。
```

## 第一張試跑

使用者不確定風格時：

1. 只產第 1 張封面。
2. 詢問是否確認此風格。
3. 確認後，依同風格自動輸出剩餘卡片。

## 一頁總結

一頁總結通常用 16:9：

- 上方：大標題
- 中間：3-5 個核心重點
- 下方：一句結論或 CTA

一頁總結不是把全部輪播卡縮在一起，而是重新濃縮成一張可獨立閱讀的總覽圖。
