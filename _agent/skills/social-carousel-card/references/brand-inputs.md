# 品牌資產與可寫回設定

## 使用者可以提供的品牌資料

越多品牌資料，輪播卡規劃越準：

- 品牌定位
- 目標受眾
- 常用語、慣用詞、禁用詞
- 口吻：務實、溫柔、專業、犀利、陪伴型等
- 價值主張
- 引流方式：留言關鍵字、私訊、報名頁、免費資源、電子報
- 常用行銷套路：痛點切入、案例拆解、教學清單、前後對照、迷思破解
- 品牌色、Logo、角色、字體、參考圖
- 範例貼文或過去表現好的輪播

## 寫回技能包

若使用者常用同一套版型或品牌規則，可以建議寫回技能包：

- 預設比例：例如全部走 4:5、全部走正方形、或改成 4:6。
- 預設張數：例如 Threads 3-5 張、IG 7-10 張。
- 預設語氣：例如「務實、少廢話、像顧問交付」。
- 預設 CTA：例如「留言關鍵字」「私訊我」「報名七月份行銷課程」。
- 禁用規則：例如不要過度誇張、不要學術腔、不要把文字塞滿。

## 最佳輸入

最穩的流程是：

1. 使用者先把文章或貼文文案寫到基本定稿。
2. 再交給本技能包拆成輪播卡架構。
3. 使用者確認卡片規劃。
4. 最後逐張生成。

如果文案還不成熟，先提醒：現在可以先做「輪播策略草稿」，但不建議直接進生圖。
