# Happiness Is Built by Habits｜幸福由習慣建立 `[書]`

納瓦爾NavalRavikant

---

我在過去五年中最令人驚訝的發現是平靜和幸福是技能。這些不是你天生就有的東西。是的，有遺傳範圍。很多是來自你的環境的條件化，但你可以解除條件化並重新條件化自己。

你可以隨著時間增加你的幸福，它從相信你可以做到開始。

這是一種技能。就像營養是一種技能，節食是一種技能，鍛煉是一種技能，賺錢是一種技能，遇見女孩和男孩是一種技能，擁有良好關係是一種技能，甚至愛也是一種技能。它從意識到它們是你可以學習的技能開始。當你把你的意圖和專注放在它上面時，世界可以變得更好。

工作時，讓自己被比你更成功的人包圍。

玩樂時，讓自己被比你更快樂的人包圍。

幸福是什麼類型的技能？

這全都是試錯。你只是看什麼有效。你可以嘗試坐姿冥想。那對你有效嗎？是譚崔冥想還是內觀冥想？是十天的閉關還是二十分鐘就夠了？

好的。那些都沒用。但如果我嘗試瑜伽呢？如果我風箏衝浪呢？如果我去賽車呢？烹飪怎麼樣？那會讓我禪嗎？你字面上必須嘗試所有這些東西，直到你找到對你有效的東西。

當涉及心靈的藥物時，安慰劑效應是百分之百有效的。當涉及你的心靈時，你想要積極傾向，而不是難以置信。如果它是完全內在的，你應該有積極的心態。

例如，我在閱讀埃克哈特·托勒的《當下的力量》，對於不信教的人來說，這是一個奇妙的臨在介紹。他向你展示最重要的事情是臨在，並一遍又一遍地強調它，直到你明白。

他寫了關於這個身體能量練習。你躺下，你感受到能量在你身體周圍移動。在那一點上，舊的我會放下書說「好吧，那是廢話。」但新的我說「好吧，如果我相信它，也許它會有效。」我以積極的心態進入它。我躺下並嘗試了冥想。你知道嗎？感覺真的很好。

有人如何建立幸福的技能？

你可以建立好習慣。不喝酒會讓你的情緒更穩定。不吃糖會讓你的情緒更穩定。不上 Facebook、Snapchat 或 Twitter 會讓你的情緒更穩定。玩電子遊戲會讓你在短期內更快樂——我曾經是一個狂熱的遊戲玩家——但從長遠來看，它可能會毀掉你的幸福。你被餵食多巴胺，並以這些小的不可控制的方式從你身上撤出多巴胺。咖啡因是另一個你用短期換取長期的例子。

基本上，你必須在你的生活中用好習慣替換你的無意識壞習慣，承諾成為一個更快樂的人。歸根結底，你是你的習慣和你花最多時間在一起的人的組合。

當我們是孩子時，我們有很少的習慣。隨著時間的推移，我們學習我們不應該做的事情。我們變得自我意識。我們開始形成習慣和例行公事。

隨著年齡增長變得更快樂的人和不快樂的人之間的許多區別可以用他們發展了什麼習慣來解釋。它們是會增加你長期幸福而不是短期幸福的習慣嗎？你是否讓自己被通常積極和樂觀的人包圍？這些關係是低維護的嗎？你欣賞和尊重但不嫉妒他們嗎？

有「五隻黑猩猩理論」，你可以透過一隻黑猩猩最常與之一起出去玩的五隻黑猩猩來預測它的行為。我認為這也適用於人類。也許說你應該非常明智地選擇你的朋友在政治上是不正確的。但你不應該根據你住在誰旁邊或你碰巧與誰一起工作而隨意選擇他們。最快樂和最樂觀的人選擇正確的五隻黑猩猩。[8]

處理衝突的第一條規則是：不要與不斷參與衝突的人一起出去玩。我對任何不可持續的東西或甚至難以維持的東西都不感興趣，包括困難的關係。[5]

如果你看不到自己與某人一起工作一生，就不要與他們工作一天。

我有一個朋友，一個波斯人叫 Behzad。他就是熱愛生活，他沒有時間給任何不快樂的人。

如果你問 Behzad 他的秘訣是什麼？他只會抬頭說「停止問為什麼，開始說哇。」世界是一個如此驚人的地方。作為人類，我們習慣於把一切視為理所當然。就像你和我現在正在做的事情。我們坐在室內，穿著衣服，吃得很飽，並透過空間和時間相互交流。我們現在應該是兩隻猴子坐在叢林裡看太陽下山，問自己我們要在哪裡睡覺。

當我們得到某樣東西時，我們假設世界欠我們的。如果你臨在，你會意識到在任何時候我們周圍有多少禮物和多少豐富。這真的就是你需要做的全部。我現在在這裡，我有所有這些令人難以置信的東西可供我使用。[8]

快樂最重要的訣竅是意識到幸福是你發展的技能和你做出的選擇。你選擇快樂，然後你為它工作。這就像鍛煉肌肉。這就像減肥。這就像在你的工作中成功。這就像學習微積分。

你決定它對你很重要。你把它放在其他一切之上。你閱讀關於這個主題的一切。[7]

# Happiness Habits｜幸福習慣 `[書]`

納瓦爾NavalRavikant

---

我有一系列我用來試圖在當下更快樂的技巧。起初，它們是愚蠢和困難的，需要很多注意力，但現在其中一些已經成為第二天性。透過虔誠地做它們，我設法相當多地提高了我的幸福水平。

明顯的一個是冥想——內觀冥想。為此努力朝向一個特定的目的，那就是試圖理解我的心靈如何運作。[7]

只是在每一刻都非常有意識。如果我發現自己在評判某人，我可以停下來對自己說「這個的積極詮釋是什麼？」我以前會對事情感到惱火。現在我總是尋找它的積極面。以前需要理性的努力。以前需要幾秒鐘才能想出一個積極的。現在我可以在亞秒內做到。[7]

我試著讓更多陽光照在我的皮膚上。我抬頭微笑。[7]

每次你發現自己渴望某樣東西時，說「這對我來說真的那麼重要，除非這按我的方式進行，否則我會不快樂嗎？」你會發現絕大多數事情根本不是真的。[7]

我認為戒掉咖啡因讓我更快樂。它讓我成為一個更穩定的人。[7]

我認為每天鍛煉讓我更快樂。如果你有身體的平靜，更容易有心靈的平靜。[7]

你評判得越多，你就越分離自己。你會在瞬間感覺良好，因為你對自己感覺良好，認為你比某人更好。後來，你會感到孤獨。然後，你到處看到消極性。世界只是把你自己的感覺反射回給你。[77]

告訴你的朋友你是一個快樂的人。然後，你會被迫符合它。你會有一致性偏見。你必須達到它。你的朋友會期望你是一個快樂的人。[5]

透過最小化你對這三個智慧型手機應用程式的使用來恢復時間和幸福：電話、日曆和鬧鐘。[11]

你擁有的秘密越多，你就越不快樂。[11]

陷入低潮？使用冥想、音樂和運動重設你的情緒。然後選擇一條新路徑，為一天的其餘時間投入情緒能量。[11]

享樂適應對人造事物（汽車、房子、衣服、錢）比對自然事物（食物、性、運動）更強大。[11]

沒有例外——所有螢幕活動都與較少的幸福相關，所有非螢幕活動都與更多的幸福相關。[11]

一個個人指標：一天中有多少時間花在出於義務而不是出於興趣做事情上？[11]

讓你焦慮和憤怒是新聞的工作。但其潛在的科學、經濟、教育和衝突趨勢是積極的。保持樂觀。[11]

政治、學術界和社會地位都是零和遊戲。

正和遊戲創造積極的人。[11]

在不使用藥物的情況下增加大腦中的血清素：陽光、運動、積極思考和色氨酸。[11]

## 改變習慣

選擇一件事。培養慾望。視覺化它。

計劃一條可持續的道路。

識別需求、觸發因素和替代品。

告訴你的朋友。

一絲不苟地追蹤。

自律是通向新自我形象的橋樑。

烘焙新的自我形象。這就是你——現在。[11]

首先，你知道它。然後，你理解它。然後，你可以解釋它。然後，你可以感受它。最後，你就是它。

# Find Happiness in Acceptance｜在接受中找到幸福 `[書]`

納瓦爾NavalRavikant

---

在生活中的任何情況下，你總是有三個選擇：你可以改變它，你可以接受它，或者你可以離開它。

如果你想改變它，那麼它是一個慾望。它會讓你痛苦，直到你成功改變它。所以不要選擇太多那些。在你生活中的任何給定時間選擇一個大慾望，給自己目的和動力。

為什麼不是兩個？

你會分心。

即使一個也夠難了。平靜來自於讓你的心靈清除思想。很多清晰來自於活在當下時刻。如果你在想「我需要做這個。我想要那個。這必須改變」，就很難活在當下時刻。[8]

你總是有三個選擇：你可以改變它，你可以接受它，或者你可以離開它。不是一個好選擇的是坐在那裡希望你會改變它但不改變它，希望你可以離開它但不離開它並且不接受它。那種掙扎或厭惡對我們大部分的痛苦負責。我在腦海中對自己使用最多的短語可能只是一個詞：「接受。」[5]
