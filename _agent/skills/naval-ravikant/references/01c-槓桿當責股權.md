# Take on Accountability｜承擔當責 `[書]`

納瓦爾NavalRavikant

---

在你自己的名義下擁抱當責並承擔商業風險。社會會用責任、股權和槓桿來獎勵你。

要致富，你需要槓桿。槓桿來自勞力、來自資本，或者它可以透過程式碼或媒體來。但大多數這些，比如勞力和資本，人們必須給你。對於勞力，必須有人追隨你。對於資本，必須有人給你錢、資產來管理，或機器。

所以要得到這些東西，你必須建立信用，而且你必須盡可能在你自己的名義下做這件事，這是有風險的。所以，當責是一把雙面刃。它讓你在事情順利時獲得榮譽，在事情糟糕時承擔失敗的衝擊。[78]

明確的當責很重要。沒有當責，你就沒有激勵。沒有當責，你就無法建立信用。但你要承擔風險。你冒著失敗的風險。你冒著在自己名義下被羞辱的風險。你冒著在自己名義下失敗的風險。

幸運的是，在現代社會，不再有債務人監獄，人們不會因為失去別人的錢而被監禁或處決，但我們仍然有社會性的硬連線，不在公眾面前以我們自己的名義失敗。那些有能力在公眾面前以自己的名義失敗的人實際上獲得了很多權力。

我給你一個個人軼事。直到大約 2013、2014 年，我的公眾形象完全圍繞著新創公司和投資。只有在 2014、2015 年左右，我才開始談論哲學和心理學的事情以及更廣泛的事情。這讓我有點緊張，因為我是用自己的名義在做這件事。業界肯定有人透過後門給我發訊息，比如「你在做什麼？你正在結束你的職業生涯。這很愚蠢。」

我只是順其自然。我冒了風險。加密貨幣也一樣。早期，我冒了風險。但當你把你的名字放出去時，你在某些事情上冒風險。你也會收穫回報。你會得到好處。[78]

在過去，船長應該與船共存亡。如果船正在下沉，那麼字面上最後一個下船的人是船長。當責確實伴隨著真正的風險，但我們談論的是商業背景。

這裡的風險可能是你可能是最後一個拿回資本的人。你會是最後一個因你的時間而得到報酬的人。你投入的時間、你投入公司的資本，這些都有風險。[78]

意識到在現代社會，下行風險並不那麼大。即使是個人破產也可以在良好的生態系統中清除債務。我最熟悉矽谷，但一般來說，只要你誠實並做出了高正直的努力，人們會原諒失敗。

在失敗方面真的沒有太多要害怕的，所以人們應該承擔比他們現在更多的當責。[78]

# Build or Buy Equity in a Business｜建立或購買企業股權 `[書]`

納瓦爾NavalRavikant

---

如果你不擁有一個企業的一部分，你就沒有通往財務自由的道路。

為什麼擁有企業股權對致富很重要？

這是所有權與工資工作的對比。如果你是因為出租你的時間而得到報酬，即使是律師和醫生，你可以賺一些錢，但你不會賺到能給你財務自由的錢。你不會有被動收入，讓企業在你度假時為你賺錢。[10]

這可能是最重要的觀點之一。人們似乎認為你可以透過工作來創造財富——賺錢。這可能不會成功。原因有很多。

沒有所有權，你的投入與你的產出非常緊密相連。在幾乎所有薪水工作中，即使是像律師或醫生這樣每小時工資很高的工作，你仍然在投入時間，而且你投入的每一小時都會得到報酬。

沒有所有權，當你睡覺時，你沒有在賺錢。當你退休時，你沒有在賺錢。當你度假時，你沒有在賺錢。而且你無法非線性地賺錢。

如果你看看甚至那些真正變富的醫生（比如真的很富），那是因為他們開了一個企業。他們開了一個私人診所。私人診所建立了一個品牌，品牌吸引人。或者他們建立了某種醫療設備、程序或具有智慧財產權的流程。

本質上，你是為別人工作，而那個人承擔了風險，有當責、智慧財產權和品牌。他們不會付你足夠的錢。他們會付你他們必須付的最低限度，才能讓你做他們的工作。那可以是一個很高的最低限度，但它仍然不會是真正的財富，你退休了但仍在賺錢。[78]

擁有公司股權基本上意味著你擁有上行空間。當你擁有債務時，你擁有有保證的收入流，你擁有下行風險。你想要擁有股權。如果你不擁有企業的股權，你賺錢的機會非常渺茫。

你必須努力到你可以擁有企業股權的地步。你可以作為小股東擁有股權，你買了股票。你也可以作為所有者擁有它，你創辦了公司。所有權真的很重要。[10]

每個真正賺錢的人在某個時候都擁有產品、企業或某種智慧財產權的一部分。這可以透過股票選擇權來實現，如果你在科技公司工作。那是一個很好的開始方式。

但通常，真正的財富是透過創辦自己的公司或甚至透過投資來創造的。在投資公司，他們購買股權。這些是通往財富的道路。它不是透過小時來的。[78]

# Find a Position of Leverage｜找到槓桿的位置 `[書]`

納瓦爾NavalRavikant

---

我們生活在一個無限槓桿的時代，對真正智識好奇心的經濟獎勵從未如此之高。[11]追隨你真正的智識好奇心是比追隨當下賺錢的東西更好的職業基礎。[11]

只有你知道或只有一小部分人知道的知識，奇怪的是，會來自你的熱情和你的愛好。如果你有圍繞智識好奇心的愛好，你更有可能發展這些熱情。[1]

如果它現在娛樂你，但總有一天會讓你感到無聊，那它就是分心。繼續尋找。

我真的只想為它們本身而做事情。這是藝術的一個定義。無論是商業、運動、浪漫、友誼，無論什麼，我認為生命的意義就是為事情本身而做事情。諷刺的是，當你為事情本身而做事情時，你會創造出你最好的作品。即使你只是試圖賺錢，你實際上會最成功。

我為自己創造最多財富的那一年，實際上是我工作最不努力、最不關心未來的一年。我主要是為了純粹的樂趣而做事情。我基本上告訴人們「我退休了，我不工作。」然後，我有時間做我面前最有價值的專案。透過為事情本身而做事情，我把它們做到了最好。[74]

你越不想要某樣東西，你越不去想它，你越不沉迷於它，你就越會以自然的方式去做它。你越會為自己而做它。你會用你擅長的方式去做它，而且你會堅持下去。你周圍的人會看到你的工作品質更高。[1]

追隨你的智識好奇心，而不是當下「熱門」的東西。如果你的好奇心曾經引導你到一個社會最終想去的地方，你會得到極高的報酬。[3]

你更有可能擁有社會還不知道如何訓練其他人去做的技能。如果有人可以訓練其他人如何做某事，那麼他們可以取代你。如果他們可以取代你，那麼他們就不必付給你很多錢。你想要知道如何做其他人在當前時期不知道如何做的事情，當這些技能有需求時。[1]

如果他們可以訓練你去做，那麼最終他們會訓練電腦去做。

你因為給予社會它想要但不知道如何在其他地方獲得的東西而得到社會的獎勵。很多人認為你可以去學校學習如何賺錢，但現實是，沒有一種叫做「商業」的技能。[1]

想想社會想要但還不知道如何獲得的產品或服務是什麼。你想成為那個提供它並大規模提供它的人。這真的是如何賺錢的挑戰。

現在，問題是在任何「它」是什麼上變得擅長。它從一代移動到另一代，但其中很多恰好發生在科技領域。

你在等待你的時刻，當世界上出現某樣東西時，他們需要一個技能組合，而你具有獨特的資格。與此同時，你在 Twitter、YouTube 上建立你的品牌，並透過提供免費工作。你為自己建立名聲，並在過程中承擔一些風險。當是時候把握機會時，你可以用槓桿來做——最大的槓桿。[1]

有三大類槓桿：

一種槓桿形式是勞力——其他人為你工作。這是最古老的槓桿形式，實際上在現代世界不是一個很好的形式。[1]我會說這是你可能使用的最糟糕的槓桿形式。管理其他人非常混亂。它需要極大的領導技能。你離暴動或被暴民吃掉或撕成碎片只有一步之遙。[78]

金錢作為槓桿的一種形式是好的。它意味著每次你做決定時，你用金錢倍增它。[1]資本是一種更難使用的槓桿形式。它更現代。這是人們在上個世紀用來變得極其富有的形式。它可能一直是上個世紀的主要槓桿形式。

你可以透過尋找最富有的人來看到這一點。是銀行家、腐敗國家印鈔票的政客，本質上是到處移動大量金錢的人。如果你看看非常大的公司的頂端，在科技公司之外，在許多、許多大型老公司中，執行長的工作真的是一個財務工作。

它的規模非常、非常好。如果你擅長管理資本，你可以比管理更多人容易得多地管理越來越多的資本。[78]

最後一種槓桿形式是全新的——最民主的形式。它是：「沒有邊際複製成本的產品。」

這包括書籍、媒體、電影和程式碼。程式碼可能是最強大的無需許可的槓桿形式。你只需要一台電腦——你不需要任何人的許可。[1]

忘記富人對窮人，白領對藍領。現在是有槓桿的對無槓桿的。

最有趣和最重要的槓桿形式是沒有邊際複製成本的產品這個想法。這是新的槓桿形式。這是在過去幾百年才發明的。它始於印刷機。它隨著廣播媒體加速，現在隨著網路和程式碼真正爆發了。現在，你可以在不涉及其他人且不需要其他人的錢的情況下倍增你的努力。

這本書是一種槓桿形式。很久以前，我必須坐在演講廳裡，親自向你們每個人演講。我可能只能接觸到幾百人，然後就結束了。[78]

這種最新的槓桿形式是所有新財富的來源，所有新億萬富翁。上一代人，財富是由資本創造的。創造財富的人是世界上的華倫·巴菲特（Warren Buffetts）。

但新一代的財富都是透過程式碼或媒體創造的。喬·羅根（Joe Rogan）從他的播客中每年賺 5000 萬到 1 億美元。你會有 PewDiePie。我不知道他賺了多少錢，但他比新聞更大。當然，還有傑夫·貝佐斯（Jeff Bezos）、馬克·祖克柏（Mark Zuckerberg）、賴瑞·佩吉（Larry Page）、謝爾蓋·布林（Sergey Brin）、比爾·蓋茲（Bill Gates）和史蒂夫·賈伯斯（Steve Jobs）。他們的財富都是基於程式碼的槓桿。[78]

關於新形式的槓桿，可能最有趣的事情是它們是無需許可的。它們不需要其他人的許可讓你使用它們或成功。對於勞力槓桿，必須有人決定追隨你。對於資本槓桿，必須有人給你錢投資或變成產品。

程式設計、寫書、錄製播客、發推特、製作 YouTube 影片——這些事情都是無需許可的。你不需要任何人的許可去做它們，這就是為什麼它們非常平等。它們是槓桿的偉大平等器。[78]例如，現在每個偉大的軟體開發者，在他或她睡覺時，都有一支機器人軍隊為他工作，在他們寫完程式碼之後，它在夜間不斷運轉。[78]

你永遠不會透過出租你的時間致富。

每當你能在生活中，優化獨立性而不是薪水。如果你有獨立性，並且你對你的產出負責，而不是你的投入——那就是夢想。[10]

人類在沒有槓桿的社會中進化。如果我為你砍木頭或為你挑水，你知道投入八小時大約等於八小時的產出。現在我們發明了槓桿——透過資本、合作、科技、生產力，所有這些手段。我們生活在一個槓桿的時代。作為一個工人，你想要盡可能有槓桿，這樣你就能在沒有那麼多時間或體力努力的情況下產生巨大的影響。

一個有槓桿的工人可以超越一個沒有槓桿的工人一千倍或一萬倍的產出。對於有槓桿的工人來說，判斷力遠比他們投入多少時間或他們工作多努力重要得多。

忘記 10 倍程式設計師。1000 倍程式設計師真的存在，我們只是沒有完全承認它。參見 @ID_AA_Carmack、@notch、中本聰等。

例如，一個好的軟體工程師，只需透過寫正確的小段程式碼和創建正確的小應用程式，就可以為公司創造價值五億美元。但十個工程師工作十倍努力，只是因為他們選擇了錯誤的模型、錯誤的產品、錯誤的寫法，或放入了錯誤的病毒式迴圈，基本上浪費了他們的時間。投入與產出不匹配，尤其是對於有槓桿的工人。

你在生活中想要的是控制你的時間。你想進入一個有槓桿的工作，在那裡你控制自己的時間，並且你被追蹤產出。如果你做了一些令人難以置信的事情來推動業務，他們必須付錢給你。特別是如果他們不知道你是怎麼做的，因為它是你的沉迷、你的技能或你的天生能力所固有的，他們將不得不繼續付錢給你去做。

如果你有專屬知識，你有當責，你有槓桿；他們必須付你應得的。如果他們付你應得的，那麼你可以拿回你的時間——你可以超高效。你不是為了開會而開會，你不是試圖給其他人留下深刻印象，你不是寫東西讓它看起來像你做了工作。你關心的只是實際工作本身。

當你只做實際工作本身時，你會更有生產力，更有效率。你會在你感覺喜歡時工作——當你精力充沛時——你不會在精力低落時試圖掙扎。你會拿回你的時間。

四十小時工作週是工業時代的遺物。知識工作者的功能就像運動員——訓練和衝刺，然後休息和重新評估。

銷售是一個例子——特別是非常高端的銷售。如果你是一個房地產經紀人在那裡賣房子，這不一定是一個好工作。它非常擁擠。但如果你是一個頂級的房地產經紀人，你知道如何行銷自己，你知道如何賣房子，你可能可以在十分之一的時間內賣出 500 萬美元的豪宅，而其他人正在努力賣出 10 萬美元的公寓或共管公寓。房地產經紀人是一個投入和產出脫節的工作。

建立任何產品和銷售任何產品都符合這個描述。從根本上說，還有什麼呢？你不一定想要的地方是支援角色，比如客戶服務。在客戶服務中，不幸的是，投入和產出彼此相對接近，你投入的時間很重要。[10]

工具和槓桿創造了投入和產出之間的這種脫節。一個職業的創造力成分越高，投入和產出越有可能脫節。如果你看的職業中你的投入和產出高度相連，那麼在那個過程中為自己創造財富和賺錢將非常困難。[78]

如果你想成為一個偉大的科技公司的一部分，那麼你需要能夠銷售或建造。如果你不做其中之一，那就學習。

學習銷售，學習建造。如果你兩者都能做，你將勢不可擋。

這是兩個非常廣泛的類別。一個是建造產品。這很難，而且是多變量的。它可以包括設計；它可以包括開發；它可以包括製造、物流、採購；它甚至可以是設計和營運服務。它有許多、許多定義。

但在每個行業中，都有一個建造者的定義。在我們的科技行業，它是技術長，它是程式設計師，它是軟體工程師或硬體工程師。但即使在洗衣業，它可能是建立洗衣服務的人，讓火車準時運行的人，確保所有衣服在正確的時間到達正確地點的人，等等。

另一邊是銷售。同樣，銷售有一個非常廣泛的定義。銷售不一定只是意味著向個別客戶銷售，但它可以意味著行銷，它可以意味著溝通，它可以意味著招募，它可以意味著籌款，它可以意味著激勵人們，它可以是做公關。它是一個廣泛的傘狀類別。[78]

用你的頭腦賺錢，而不是你的時間。

# Get Paid for Your Judgment｜因判斷力獲得報酬 `[書]`

納瓦爾NavalRavikant

---

選擇你進入什麼樣的工作、職業或領域，以及你願意從雇主那裡接受什麼樣的交易，會給你更多的自由時間。然後，你就不必太擔心時間管理。我希望純粹因為我的判斷力而得到報酬，而不是因為任何工作。我想要一個機器人、資本或電腦來做工作，但我想因我的判斷力而得到報酬。[1]

我認為每個人都應該渴望了解某些事情，並因我們獨特的知識而得到報酬。我們在業務中擁有盡可能多的槓桿，無論是透過機器人還是電腦或其他什麼。然後，我們可以成為自己時間的主人，因為我們只是被追蹤產出而不是投入。

想像一下，有人出現了，他們明顯擁有稍微好一點的判斷力。他們 85% 的時間是對的，而不是 75%。你會付他們 5000 萬美元、1 億美元、2 億美元，不管需要多少，因為在駕駛一艘價值 1000 億美元的船時，10% 更好的判斷力是非常有價值的。執行長的薪酬很高是因為他們的槓桿。判斷力和能力的微小差異真的被放大了。[2]

已展示的判斷力——圍繞判斷力的信用——是如此關鍵。華倫·巴菲特在這裡獲勝，因為他有巨大的信用。他一直非常負責任。他在公眾領域一次又一次地是對的。他建立了非常高正直的聲譽，所以你可以信任他。人們會在他身後投入無限的槓桿，因為他的判斷力。沒有人問他工作有多努力。沒有人問他什麼時候醒來或什麼時候睡覺。他們說：「華倫，做你的事吧。」

判斷力——特別是已展示的判斷力，具有高度當責和清晰的記錄——是關鍵的。[78]

我們浪費時間在短期思考和忙碌工作上。華倫·巴菲特花一年決定，花一天行動。那個行動持續數十年。

僅僅因為稍微好一點，比如跑四分之一英里快了幾分之一秒，有些人得到的報酬要多得多——數量級更多。槓桿進一步放大了這些差異。在槓桿時代，在你的藝術中處於極端是非常重要的。[2]

透過迭代解決。

然後透過重複獲得報酬。

## 2025 訪談 II 延伸：「Truly work for yourself」自由的味道讓人 unemployable `[2025訪談II]`

> 詳見 `01h-Life-in-the-Arena-2025訪談II.md` 第四章。「以自己的名義承擔當責、用槓桿、賺判斷力的錢」這條原則，在這集講出了一個少見的角度：**勞動本身可以是自我表達**，自由一旦嘗過就回不去。

### 為自己工作的悖論

> 「April 2 推文：When you truly work for yourself, you won't have hobbies, you won't have weekends, and you won't have vacations. But you won't have work either.」

開始為自己工作 → 沒 9-to-5、沒辦公室、沒人告訴你做什麼、沒 playbook。**同時也關不掉**——你就是事業，你就是產品，工作關不掉。這是 entrepreneur 的詛咒。

但好處是：方向對、人對、做對了，**它根本不像工作**。你只用產出衡量自己，標準是你訂的。極度釋放、極度興奮。

### 自由的味道讓人 unemployable

> 「A taste of freedom can make you unemployable.」

一旦嘗過了，回不去 9-to-5、回不去 boss、回不去 check-in。這是**單向門**。不只是辭職的人會這樣——小團隊裡自驅型成員也會。看似巨大的負面（沒週末沒假期），但**工作的時候根本不像在工作**——這就是回報。

### 雙重解讀：勞動是自我表達

「Truly work for yourself」還有一層意思：**你的勞動是你是誰的表達**。Self-expression。這不容易找到——你必須做你獨特最擅長、跟你的本質對齊的事，那會給你 authenticity、specific knowledge、irreplaceability。

而**找到那個東西的唯一方法是行動**——也就是「以自己的名義承擔當責」的最終目的。當責不只是換槓桿的代價，**還是發現「你是誰」的方法**。

## 2026 訪談延伸：程式碼槓桿走到極致 `[2026訪談]`

> 詳見 `01g-Vibe-Coding-2026訪談.md`。以下是「程式碼與媒體是無需許可的槓桿」這個論點，在 AI coding agent 時代的最新延伸。

### 純軟體不可投資（uninvestable）

我說過程式碼是新富階層背後的力量。但 2026 年我要更新一個結論：**純軟體現在不可投資。**

理由：(1) 別人今天就 hack 得出來；(2) Coding agent 在一年內甚至更短，就能寫出有架構、可擴展的軟體。

VC 現在要找的是：硬體、網路效應、AI 模型訓練。**訓練 AI 模型是新時代的「寫軟體」**——直到 auto-research、auto-training 起飛為止。

這不矛盾——程式碼槓桿仍然是最大的槓桿，但它已經太普及，不再是稀缺資源。稀缺的是：方向、品味、判斷力、無需許可的分發管道。

### 個人 App Store ＝ 程式碼槓桿的極致

我為自己建了一個個人 App Store：兩行描述 → Claude 建 App → 推到我手機 → 30 秒後可用。可以在晚餐時聽朋友說他想要某個 App，5 分鐘後拿給他看。

這是「無需許可的槓桿」的最新極致——不只是不需要別人投錢、不需要別人追隨你，**現在連寫程式的能力都不需要**。槓桿的門檻幾乎歸零。

### Coding Agent ＝ 完美客服

我的 App 有 bug-reporting 系統：使用者按鈕回報 → Claude 每 24 小時跑過全部 bug → 自動修，推到 side branch → 我只當最後 reviewer。

> 「如果你的客服真的完美，那個人也是超強工程師、永不疲倦、24/7、沒有 ego——寫了一大堆 code 修 bug 然後說丟就丟。」

這就是當責 + 槓桿在 AI 時代的新形態：你保留判斷力與品味（當責），把所有可機械化的工作交給 agent（極致槓桿）。

### 一兩個人公司可以 scale 到百萬使用者、賺數十億美金

Notch、中本聰、原始 Instagram、原始 WhatsApp 都發生過。**未來這個模式只會更頻繁。** 這是「擁有股權」「無需許可槓桿」「以判斷力獲得報酬」三條原則交會的最新證據。

## 2026 訪談首集延伸：抽象堆疊新層、軟體工程師更被槓桿、英文是新程式語言 `[2026訪談首集]`

完整脈絡見 `01i-If-You-Want-to-Learn-Do-2026訪談首集.md`。

### Vibe coding 是新的產品管理

Claude Code 等工具強到讓「不太寫程式或久沒寫的人」用英文當 end-to-end coding 引擎。意涵：「**Vibe coding 是新的產品管理**——你不再透過告訴一群工程師要做什麼來管產品，你在告訴電腦要做什麼。電腦不會累、沒有自尊、不會被回饋冒犯。」就像現在任何人都能拍片、做 podcast，**現在任何人都能做 App**。

### 訓練模型是新的 coding

「如果任何人都能寫 code，coding 還剩什麼？最明顯的：訓練這些模型本身。」傳統 coding 是極精準指令；AI coding 是「在一個結構裡尋找一個程式」。這是為什麼 AI 研究者拿到天文數字薪水——**他們本質上接管了 programming**。

### 傳統軟體工程師沒有死，反而是地球上最被槓桿化的人之一

軟體工程師有兩個無法被取代的優勢：
1. **他們用程式碼思考**——所有抽象都會漏（all abstractions are leaky），能 plug the leaks
2. **AI 在 data distribution 之外仍弱**——高效能 code、新架構、新問題仍要工程師下手

> 「**一個在領域知識前緣的好工程師可以把 vibe coder 玩弄於股掌之間**（run circles around vibe coders）。」

要怎麼成為那樣的人？回到老推文：「**Become the best in the world at what you do. Keep redefining what you do until this is true.**」AI 時代依然成立——你能做最強的事是無限的，總能找到一個 niche 讓你最強。

「**沒有人想要平均**——平均沒有市場。最好的 App 拿走 100% 的市場。」

### 英文是最熱的新程式語言

把 AI coding 想成**抽象堆疊（abstraction stack）裡的新一層**——電晶體 → 晶片 → 組合語言 → C → C++ → Python → 自然語言。

「我念書時主要寫 C，後來 C++ 出來、Python 出來，每一層都更抽象。**現在你真的能用英文寫程式**。這是 emergent property，是大跳躍不是小改進。」

> 「這就是 Andre Karpathy 說『**English is the hottest new programming language**』的意思。」

對「結構化思考、會講話、知道電腦架構」的人——你能用結構化的英文精準指定你要的東西。**不需要再學什麼 esoteric programming command**。
