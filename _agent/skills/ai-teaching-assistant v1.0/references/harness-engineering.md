# 延伸閱讀：駕馭工程、LLM Wiki、Hermes Agent

想深入了解 AI 辦公室背後的理論基礎，看這份。

## 使用時機

- 使用者已經用過一陣子，開始問「為什麼要這樣設計」
- 使用者對 Agent 架構、知識管理技術感興趣
- 使用者是已經會寫程式、或熟悉 AI 工具的人

---

## 零、快速比喻：用賽馬一口氣說清楚四個詞

這四個詞常常被混著說，用賽馬類比可以一次說清楚：

| 詞 | 賽馬比喻 | 一句話定義 |
|---|---|---|
| **AI（大語言模型）** | 馬的大腦 | 負責運算、推理、理解語言。沒接上外部工具前，它只能「開口說話」 |
| **Agent（代理）** | 一匹馬 | 把大腦接上了手腳。除了說話，還能動手：搜尋網路、讀檔案、執行程式 |
| **Harness Engineering（駕馭工程）** | 韁繩 + 路線 + 檢查點 | 控制馬怎麼跑、往哪裡跑、什麼時候停下來。讓 Agent 不亂跑，乖乖完成任務 |
| **Agentic AI（代理式 AI）** | 一整場賽馬比賽 | 馬場（硬體環境）+ 在跑的馬（Agent）+ 賽制規則（任務系統）。一套能自己跑完任務的完整系統 |

**關鍵差異**：
- Agent 是「能動手的 AI」
- Harness Engineering 是「讓 Agent 可控的方法」
- Agentic AI 是「整套自動完成任務的系統設計」——不只讓 Agent 可控，而是設計一個能自己跑完的流程

**對使用者的意義**：
你下載 AI 辦公室，就是在使用一套 Agentic AI 系統。你的每個技能包就是「賽制規則」；AI 助理是「馬」；你指令的方式就是「駕馭工程」。

## 一、Harness Engineering（駕馭工程）

英文 Harness 是韁繩的意思。本質上就是在駕馭 AI。

### OpenAI 2026 年的定義

> Agent = AI 模型（大腦）+ 我們如何控制這顆大腦的工程（駕馭工程）

大腦接 ChatGPT 還是 Claude 都可以，但接完之後怎麼指揮它運作、不讓它失控，就是駕馭工程。

### 非工程師能做的駕馭工程

不會寫程式沒關係。能做的是：

- **設計環境**：資料夾結構、文件放置邏輯
- **定義意圖**：用文件寫清楚每個資料夾的用途、每個流程的規則
- **建立迴圈**：讓 AI 照著我們寫好的路徑自動跑

AI 會越來越聰明，但有些文件的特殊邏輯還是要我們自己定義。
你讓 AI 自動駕駛，它一定跑 ABC；要讓它跑 ACB，你就得告訴它。

**AI 辦公室的每個設計——三基本功、歡迎語、路由規則——都是駕馭工程的實踐。**

## 二、LLM Wiki

### 核心命題

讓你的知識庫可以**持續累積成長、更有系統地自動更新**。

### 跟傳統知識圖譜的差別

傳統知識圖譜要手動維護連結。
LLM Wiki 把這件事做成了**自動累積、自動連結、自動更新**的知識庫——
丟進新文件 D，AI 自動判斷 D 跟 A、B、C 的關聯並建立連結。

特殊案例仍建議手動補充。

### 知識連結的基本原理

每一份文件，原本是獨立的點。
AI 查找的時候，看到 A、B、C 三份，預設按 ABC 順序讀取。

但實際的工作流程可能是 ACB（先 A，突發狀況跳 C，完成後才回 B）。

你需要**告訴 AI 這次是 ACB**。

兩種方法：
1. **時間戳**：提示
2. **連結語法**：指定（更精確）

### 跨系統連結

假設你有兩個系統：
- 系統一：A、B、C
- 系統二：甲、乙、丙

如果 B 跟丙的底層邏輯幾乎一樣，可以在 B 的文件頭部寫：
> 請參照系統二的丙，因為底層邏輯幾乎相同。

AI 查 B 的時候，就知道去丙那邊也參考一下。

**這就是知識圖譜——文件之間有方向性的連結。**

## 三、Hermes Agent

### 核心特色

**「會陪著使用者成長的 Agent，會自我進化。」**

「自我進化」的實際意思不是模型越來越聰明，而是：
- 使用者交代的事情，它會記住
- 使用者沒有交代的事情，它也會自己分析你的喜好並記住
- 處理完複雜工作流程後，會把這次做得順的地方和重複執行的步驟，**自己整理成技能包（Skill）**
- 技能包還會自動更新

### 比喻

就像一個超認真做小本子的助理，你說的他記，你沒說的他也偷偷記，需要的時候主動給你驚喜。

### 跟 AI 辦公室的關係

AI 辦公室的「任務結束後主動提問」機制（要不要寫工作日記、要不要做技能包）就是 Hermes Agent 概念的簡化版。

江江教練的策略：
> 我每次有新工具出來，不是整個搬家，而是看它優點在哪，把優點吸收進我的系統。

## 四、三個概念串起來

- **駕馭工程**告訴你：不會寫程式也能駕馭 AI
- **LLM Wiki** 告訴你：知識庫不只是存檔，還能自動串連
- **Hermes Agent** 告訴你：AI 會陪著你成長，自我進化

AI 辦公室 = 駕馭工程 + LLM Wiki 的簡化入門版 + Hermes Agent 概念的主動提問機制

你現在用的每一個動作，背後都是這三個概念的落地。
