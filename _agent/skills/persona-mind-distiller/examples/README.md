4. 雙語雙倉——中英文版同步維護

## 怎麼使用範例

**錯誤做法**：直接複製 Naval 的設定檔，把名字改掉。

**正確做法**：

1. 先讀 Naval 的 10 區塊，看每個區塊**內容的密度跟具體度**
2. 對照自己提煉的人格，問「我有寫到這個密度嗎？」
3. 看 Naval 的思維引擎六個子模組，學**怎麼組織思考方式**
4. 看 Naval 的雙檢索，學**問句怎麼反向索引**

範例是用來**校準品質的尺**，不是模板。
