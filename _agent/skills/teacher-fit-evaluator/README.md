# 這個老師適不適合我

> 用三節奏 + 五維度，幫你判斷一個老師（或課程／PDF／論文／GitHub Repo）值不值得學。
> 核心不是「這老師厲不厲害」，是「**跟我的工作流合不合**」。

## 為什麼需要這個技能包

每天有人推薦你新老師、新課程、新論文、新 Repo：

> 「這個老師超強！」
> 「這堂課必上！」
> 「這個 Repo 一定要 star！」

但你打開連結看半天，還是不知道：

- **跟我合不合？**
- **接得進我現在的工作流嗎？**
- **學完我能拿來做什麼？**

這個技能包用 AI 幫你做**個人化判斷**——不是公評分卡，是專屬於你工作流的契合度檢查。

## 三節奏

### 🗺️ 節奏一｜建立你的學習地圖（第一次跑一次）

技能包問你 8 個問題（圍繞「你的工作流」），畫出你的學習地圖：

1. 主要工作 / 角色
2. 工作流 / 工作節奏
3. 最卡住的環節
4. 已掌握的技能
5. 想精進的環節
6. 學習偏好
7. 時間預算
8. 判斷準則

存成 `my-learning-map.md`，之後評估老師時 AI 自動讀。

> **副產品**：這份地圖不只用來評估老師，本身就是你的工作流自我認識。可以拿來介紹自己、跟新客戶開場、定期回顧成長軌跡。

### ⚡ 節奏二｜快速評估老師（不耗算力）

提供老師／資源的基本資料（網址、簡介、課綱），AI 快速比對你的學習地圖：
- **契合度**（20）— 跟你工作流合不合？
- **產出力**（20）— 學完能做出什麼？

結尾固定問：「要不要進深度評估？」

### 🔍 節奏三｜深度評估老師（要你確認、耗算力）

兩個子節奏，可單跑、可全跑：

- **3A 可信度查**：你提供老師社群帳號，AI 查實戰可信度與願景完整度
  - 套用 **FTC Coaching Scam 警訊清單**做扣分檢查
  - 產出：可信度（20）

- **3B 橫向對比**：找 2-3 個類似老師／資源做表格對照
  - 產出：投資產出比（20）+ 不可替代性（20）

## 五維度評分卡（總分 100）

| # | 維度 | 節奏 | 滿分 | 核心問句 |
|---|---|---|---|---|
| 1 | 契合度 | 2 | 20 | 跟我合不合？接得進我的工作流嗎？ |
| 2 | 產出力 | 2 | 20 | 學完能做出什麼？ |
| 3 | 可信度 | 3A | 20 | 這個老師可不可信？ |
| 4 | 投資產出比 | 3B | 20 | 划不划算？ |
| 5 | 不可替代性 | 3B | 20 | 有沒有更好的替代？ |

每個節奏都跑完才會給總分 100；只跑節奏 2，總分 40，未跑的維度誠實標 N/A。

## 安裝

### Claude Code / Claude Desktop



## 使用範例

### 第一次使用

```
你：幫我看看這個老師 https://example.com/teacher-x

AI：我注意到你還沒建立學習地圖。先花 5 分鐘問你 8 個問題，
畫出你的工作流地圖，之後評估老師會更精準。

[1/8] 你目前的主要工作或角色是什麼？

你：自由攝影師，接商業攝影案

AI：[2/8] 你的工作流分哪幾個階段？

...（8 個問題）

AI：學習地圖建好了，存到 my-learning-map.md。
現在開始評估這位老師...
```

### 已有學習地圖

```
你：這個課值得上嗎？[貼課綱]

AI：（讀 my-learning-map.md）
契合度 16/20、產出力 13/20，建議「繼續深查」。
要不要進節奏 3A（可信度）或 3B（橫向對比）？

你：A

AI：請給我老師的社群帳號或網站連結。

你：[貼連結]

AI：（搜尋三角度）可信度 14/20。累計 43/60。
要不要再做節奏 3B？
```

## 評估框架的學術／業界依據

五維度設計參考多個既有評估框架：

| 領域 | 框架 | 用在哪 |
|---|---|---|
| 教育學界 | Kirkpatrick 四層、CIPP、Danielson | 維度 1-2 |
| 產品評估 | Gartner Magic Quadrant、G2 Score | 維度 3-4 |
| 個人化匹配 | MCRS、Global User Profile | 學習地圖設計 |
| 學習決策 | CDC Quality Training Standards、ICF Coach Evaluation | 維度 3 |
| 可信度判斷 | SIFT 評估法、NIH Health Information Evaluation | 維度 3 |
| 風險訊號 | FTC Coaching Program Scam Warning | 維度 3 扣分項 |

完整對照與來源詳見 [references/frameworks-reference.md](references/frameworks-reference.md)。

## 設計理念

### 為什麼分三節奏？

ICF「How Do I Choose a Coach?」明確說：「**先訪談 2-3 位教練再決定**」。選擇老師應該是分階段、漸進的決策過程，不是一次到位。

| 節奏 | 時長 | 決策成本 |
|---|---|---|
| 節奏 2（快速契合度）| 30 秒到 2 分鐘 | 低 |
| 節奏 3A（深度可信度）| 2-3 分鐘 | 中 |
| 節奏 3B（橫向對比）| 3-5 分鐘 | 中高 |

### 為什麼需要學習地圖？

你的工作流是評估老師的**參考座標系**。沒有座標系，「契合度」就是空話。

學習地圖的副作用：定期更新等於做工作流自我體檢。一年下來會看到自己的成長軌跡。

### 為什麼是 100 分制？

100 分制比 60 分制好記、好溝通。每個維度 20 分，正好跟「五維度」對齊。
