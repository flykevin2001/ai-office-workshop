# 評分卡細則與輸出模板

每個維度滿分 20，總分 100。本檔定義各維度的子項、評分標準、輸出格式。

---

## 維度 1｜契合度（20）

**核心問句**：跟我合不合？接得進我的工作流嗎？

**評分節奏**：節奏 2（不耗算力）

### 子項與分數分配

| 子項 | 滿分 | 在看什麼 |
|---|---|---|
| 主題相關性 | 7 | 跟你「現在在做的事 / 工作流」對不對得上 |
| 深度匹配 | 7 | 太淺浪費／剛好／太深跟不上 |
| 系統接軌 | 6 | 能不能接到你的工作流既有環節 |

### 內部「定位四問」（評分前問自己）

借自江江原版的學習地圖定位法：

1. **找鄰居**：跟你已掌握的技能哪個最相關？
2. **找邊界**：差異在哪？
3. **判斷價值**：補了什麼缺口，還是只是重疊？
4. **找落點**：學了之後放在工作流的哪一段？

回答得越清楚，契合度分數越高。

### 各子項評分指引

**主題相關性（0-7）**
- 7：直接命中你「最卡住的環節」或「想精進的環節」
- 5：相關但偏一點（同領域但角度不同）
- 3：擦邊（同產業／同工具但主題不直接）
- 1：弱相關（要繞兩層才能用）
- 0：無關

**深度匹配（0-7）**
- 7：剛好補你目前的盲點，不太淺也不太深
- 5：略淺但有 1-2 個新點子；或略深但前 70% 跟得上
- 3：明顯偏淺（你已知 80%）或偏深（前提知識超過 50% 沒有）
- 1：嚴重不匹配
- 0：完全不對

**系統接軌（0-6）**
- 6：可以直接放進工作流的某個環節，立刻取代既有做法或補上空缺
- 4：需要改造，但改造方向清楚
- 2：要重組，可能只能參考片段
- 0：放不進現有工作流

### 輸出格式

```markdown
【契合度】X/20
- 主題相關性：X/7 — [具體理由，提到對應的工作流環節]
- 深度匹配：X/7 — [太淺/剛好/太深+理由]
- 系統接軌：X/6 — [可接到工作流的「OO 階段」，或無法接]

定位四問：
- 找鄰居：[跟已掌握的 OO 最相關]
- 找邊界：[差異在 OO]
- 判斷價值：[補了 OO 缺口 / 跟 OO 重疊]
- 找落點：[放到工作流的 OO 環節]
```

## 維度 2｜產出力（20）

**核心問句**：學完能做出什麼？

**評分節奏**：節奏 2

### 評分指引

不只看「學到知識」，要看「能做出具體成果」（借 Kirkpatrick 的 Behavior + Results 層次）。

| 分數區間 | 描述 |
|---|---|
| 17-20 | 學完一週內能做出具體產出（一篇文章／一個流程改版／一個技能） |
| 13-16 | 學完一個月內能轉化成產出，但需要消化 |
| 9-12 | 能改變思考方式，但要過幾個月才看得到產出 |
| 5-8 | 純觀念性，靠你自己整合才有產出 |
| 0-4 | 學完只是「知道了」，無具體可做之事 |

### 輸出格式

```markdown
【產出力】X/20
- 學完能做出什麼具體成果？
  1. [具體成果 1]
  2. [具體成果 2]
  3. [具體成果 3]（如有）
- 是否有「知道但用不出」的風險？[是/否+理由]
- 對應你的「判斷準則」（學習地圖第 8 題）：[符合 / 不符合]
```

## 維度 3｜可信度（20）

**核心問句**：這個老師可不可信？

**評分節奏**：節奏 3A（耗算力，要使用者確認 + 提供連結）

### 雙軸內評（合成一個分數）

**願景完整度（內評，影響 0-10）**
- 他的理論架構完不完整？想得清楚嗎？
- 看：他公開講過的核心觀點、文章、訪談
- 高分：有清楚的世界觀、方法論、體系
- 低分：散裝知識、沒有自己的觀點

**實戰可信度（內評，影響 0-10）**
- 他自己有在做嗎？做得怎樣？
- 看：他過去做出什麼成果、學員回饋、業界口碑
- 高分：有實戰案例、有作品、有人做出結果
- 低分：只是教課但自己沒做、沒實戰證據

### 加分項

每項 +1，最多 +3：

- ✅ 公開承認過自己的失敗或盲點
- ✅ 有持續更新的實戰紀錄
- ✅ 學員有具體可驗證的產出（不是匿名見證）
- ✅ 業界同行的正面評價（非業配、非交換推薦）
- ✅ 有特定領域的可驗證資歷（不是跨領域大頭銜）
- ✅ 有清楚的退費 / 不滿意機制

### 扣分項（FTC Coaching Scam 警訊清單）

每項 -2，最多 -8：

- ⚠️ 誇大收入承諾（「跟我學年薪百萬」）
- ⚠️ 保證快速致富 / 速成承諾
- ⚠️ 隱藏退款條件 / 退款流程不透明
- ⚠️ 大量 incentivized testimonials（推薦人收到回扣）
- ⚠️ 虛假認證 / 認證無法查證
- ⚠️ 過度自信，沒有保留條件
- ⚠️ 學員回饋有結構性負評（多人講同樣問題）
- ⚠️ 過去的教學承諾未兌現（公開可查）
- ⚠️ 跨領域過度延伸（健身教練教投資、設計師教 AI 商業模式）

### 輸出格式

```markdown
【可信度】X/20

願景完整度：X/10 — [具體判斷+引用 2-3 個來源]
實戰可信度：X/10 — [具體判斷+引用 2-3 個來源]

✅ 加分點：
- [具體事實 1]
- [具體事實 2]

⚠️ 扣分點 / 警訊（如有）：
- [具體事實 1]

【總分計算】
基礎分：(願景 + 實戰) = X
加分：+X
扣分：-X
最終：X/20（上下限 0-20）
```

## 維度 4｜投資產出比（20）

**核心問句**：划不划算？

**評分節奏**：節奏 3B（耗算力，要使用者確認）

### 評分指引

跟「同類選項」橫向比較，不是看絕對成本。

| 分數區間 | 描述 |
|---|---|
| 17-20 | 同類最划算（時間／金錢成本最低或產出最高） |
| 13-16 | 中上水準，比平均好 |
| 9-12 | 平均水準 |
| 5-8 | 比同類差，但不算離譜 |
| 0-4 | 明顯不划算 |

### 評估時要列的數字

- **時間成本**：總時數（含預習、上課、複習、實作）
- **金錢成本**：學費 + 教材 + 隱性成本（交通、設備）
- **預期收穫**：學完能拿到的產出（用維度 2 的結果）

### 輸出格式

```markdown
【投資產出比】X/20

成本：
- 時間：約 X 小時
- 金錢：約 X 元

跟同類比：
- 替代 1：[名稱]，成本 X，產出 X
- 替代 2：[名稱]，成本 X，產出 X
- 替代 3：[名稱]，成本 X，產出 X

定位：[同類最划算 / 中上 / 平均 / 偏貴 / 不划算]
理由：[一兩句]

對應你的「時間預算」（學習地圖第 7 題）：[匹配 / 不匹配]
```

## 維度 5｜不可替代性（20）

**核心問句**：有沒有更好的替代？

**評分節奏**：節奏 3B（搭配維度 4 一起跑）

### 評分指引

| 分數區間 | 描述 |
|---|---|
| 17-20 | 獨特性高，找不到能取代的（人或內容有獨家性） |
| 13-16 | 有些獨家點，但部分可以由替代品提供 |
| 9-12 | 有替代品，但這個有自己的特色 |
| 5-8 | 多數內容能被替代品取代 |
| 0-4 | 有明顯更好的替代品，這個沒必要 |

### 輸出格式

```markdown
【不可替代性】X/20

這個對象獨有的：
- [獨特點 1]（如果有）
- [獨特點 2]（如果有）

替代品更好的地方：
- [替代品 OO 在 XX 方面更好]（如果有）

定位：[找不到取代 / 有特色 / 普通 / 有更好替代]
```

## 最終評分卡輸出模板

依「跑了哪些節奏」決定總分滿分：

| 跑的節奏 | 滿分 | 已評維度 |
|---|---|---|
| 只節奏 2 | 40 | 1, 2 |
| 節奏 2 + 3A | 60 | 1, 2, 3 |
| 節奏 2 + 3B | 80 | 1, 2, 4, 5 |
| 全跑 | 100 | 1, 2, 3, 4, 5 |

### 完整版範本

```markdown
🎯 最終評估報告：[對象名稱]

【一句話結論】
[強烈推薦／推薦／條件性推薦／不推薦／略過]，原因 [一句]

【評分卡】（已跑節奏：[2 / 2+3A / 2+3B / 全跑]）
| 維度 | 分數 | 滿分 |
|---|---|---|
| 1. 契合度 | X | 20 |
| 2. 產出力 | X | 20 |
| 3. 可信度 | X | 20（或 N/A） |
| 4. 投資產出比 | X | 20（或 N/A） |
| 5. 不可替代性 | X | 20（或 N/A） |
| **總分** | **XX** | **XX** |

百分比：XX%

【哪裡能用、怎麼用】
1. [具體切入點 1]：可以接到工作流的 [OO 環節]
2. [具體切入點 2]：可以用在 [OO 場景]
3. [具體切入點 3]（如有）

【建議投入時間】
- [30 分鐘瀏覽 / 2 小時精讀 / 整堂課上 / 略過]

【未查的維度（如有）】
- [列出哪些維度沒跑，建議補哪一個會最有資訊增益]

【決策建議】
- [明確的去/不去/再觀察建議，附理由]

【跟你的學習地圖對照】
- 卡住點 → [是否解到]
- 想精進 → [是否補強]
- 時間預算 → [是否匹配]
- 判斷準則 → [是否符合]
```

## 百分比 → 建議對應表

| 百分比 | 建議 |
|---|---|
| 80% 以上 | 強烈推薦，馬上學 / 報名 / 讀 |
| 60-79% | 推薦，找時間吸收 |
| 40-59% | 條件性推薦／有保留 — 看是否補「未查」維度後翻盤 |
| 20-39% | 不太推薦 — 除非有特殊理由 |
| < 20% | 略過 — 把時間留給更值得的 |
