# 技能包大總管 工作日記範本

每次跑完任一模式建議留一份。下次盤點時對照看軌跡。

## 範本

```markdown
# 技能包大總管 工作日記：[模式名稱]

時間：YYYY-MM-DD HH:MM
模式：[一/二/三/四]
範圍：[掃了什麼]

## 動了什麼

- [列出實際採納的動作]
- [合併了 A + B → C]
- [拆了 D → D-core / D-extra]
- [封存了 E]
- [寫入 F SKILL.md 三條新規則]

## 沒動什麼（重要）

- [列出本來想動但決定不動的]
- [理由]

## 候選池更新

- 新增：[本次沒寫入但有潛力的規則]
- 強制寫入：[連續 3 次通過的]
- 永久丟棄：[連續 3 次沒寫入的]

## 下次注意

- [這次跑完發現需要下次處理的事]
```

## 命名建議

`YYYY-MM-DD-HHMM 技能包大總管：模式X 報告.md`

例：`2026-05-12-2200 技能包大總管：模式四 報告.md`

## 存放位置

公開版不強制位置，使用者可指定。常見選擇：

- `工作日記/` 統一存放
- 各技能包 `references/diary/` 各自保留
- 獨立資料夾 `技能包大總管日記/`

選一個就好，固定下來不要分散。
