## 使用說明

- 新增標籤前先查這份字典，確認沒有同義詞已存在
- 發現同義詞請告知 AI，啟動「整理受控詞彙」流程
- 新詞先放「待確認」區，確認後再移入對應分類
- 停用詞彙不要刪除，保留備查
